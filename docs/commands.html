This plugin provides the following commands (sorted in the order that I find them useful subjectively) that are available from the Command palette, Slash commands, or your custom hotkeys.

  • SearchSearch
    • Powerful especially when combined with Dataviewcombined with Dataview!
  • Insert proofInsert proof
  • Insert display math
    • "Properly" insert a display math $$ ... $$ even inside a callout or a blockquote.
  • Open local settings for the current noteOpen local settings for the current note
    • Also available from More options (three dots) menu or right-clicking the file explorer
  • Insert theorem calloutInsert theorem callout
    • For GUI lovers
  • Migrate from version 1Migrate from version 1
    • Don't forget to run this if you are using this plugin since Math Booster version 1!
  • Convert equation numbers in the current note to static \tag{}Convert equation numbers in the current note to static \tag{}
    • This is supposed to be no longer necessary in version 2, because equation numbers can now be successfully printed in PDF export!
